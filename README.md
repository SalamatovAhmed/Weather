Веб-приложение для просмотра актуальной информации о погоде в реальном времени.
Пользователь может ввести название города и получить данные о текущей погоде.
Используемые технологии
HTML5
CSS3
JavaScript (ES6+)
REST API
Fetch API 
Git, GitHub
В проекте используется API погоды для получения данных по названию города:
текущая температура
погодные условия
влажность и ветер
Все запросы выполняются асинхронно с использованием Fetch API
